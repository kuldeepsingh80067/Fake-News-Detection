# 📰 Fake News Detection App

An AI-powered web application that detects whether a news article is **Real or Fake** using **Machine Learning (NLP)**.

Built with ❤️ using **Streamlit**, **Scikit-learn**, and **TF-IDF Vectorization**.

---

## 🚀 Features

* 🧠 Detects fake vs real news instantly
* ✍️ User-friendly text input interface
* 📊 Confidence score for predictions
* ⚡ Fast and lightweight model
* 🌐 Deployable on Streamlit Cloud

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas & NumPy
* NLP (TF-IDF Vectorizer)

## 🧠 How It Works

1. Text is cleaned using preprocessing techniques
2. Converted into numerical format using **TF-IDF Vectorizer**
3. Passed into a **Logistic Regression model**
4. Model predicts whether the news is **Real or Fake**

---

## 📊 Model Details

* Algorithm: Logistic Regression
* Feature Extraction: TF-IDF
* Type: Supervised Machine Learning
* Output: Binary Classification (Real / Fake)
